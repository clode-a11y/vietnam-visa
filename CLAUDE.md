# CLAUDE.md

Контекст для Claude Code при работе с этим репозиторием.

## Что это за репозиторий

Учебный проект для практики с Claude Code. Люди приходят, выполняют задачи, учатся программированию.

## Структура

```
nha-trang/
├── ONBOARDING.md       # Инструкция для новичков
├── tasks/              # Задачи для выполнения
│   └── visa/           # Визовый сайт
└── docs/               # Документация проектов
    ├── rent/           # Сайт аренды жилья
    └── map/            # Туристическая карта
```

## Проекты

### 1. Визовый сайт (tasks/visa/)
Информационный сайт про визу во Вьетнам. Задачи для начинающих.

### 2. Сайт аренды (docs/rent/)
Airbnb-подобный сайт для аренды квартир в Нячанге. Основной проект.

**Ключевые фичи:**
- Запрос на осмотр квартиры
- Видеозвонок из квартиры
- Статус доступности (свободна / освободится)
- Фильтрация по районам Нячанга
- Мультиязычность (RU/EN/VN)

### 3. Туристическая карта (docs/map/)
Интерактивная карта с полезными местами для туристов в Нячанге.

**Ключевые фичи:**
- Карта с категориями мест (еда, пляжи, массаж, банкоматы...)
- Детальные карточки мест с фото и отзывами
- Пользователи могут предлагать новые места (модерация)
- Фильтры и поиск
- Геолокация

## Как работать с исполнителями

Когда человек приходит выполнять задачу:

1. Он уже прошёл ONBOARDING
2. Направь его в нужную папку tasks/
3. Помоги выполнить задачу пошагово
4. Объясняй простыми словами — это новички
