# Pages — Туристическая карта

## Структура страниц

```
/                           - Главная (карта)
/place/[slug]              - Страница места
/suggest                   - Предложить место
/admin                     - Админка (защищена паролем)
  /admin/places            - Управление местами
  /admin/places/new        - Добавить место
  /admin/places/[id]/edit  - Редактировать место
  /admin/suggestions       - Модерация предложений
  /admin/reviews           - Модерация отзывов
  /admin/categories        - Управление категориями
```

## Адаптивность

### Mobile (< 768px)
- Карта занимает весь экран
- Фильтры — горизонтальный скролл
- Popup — снизу как sheet
- Админка — бургер-меню

### Tablet (768px - 1024px)
- Карта с sidebar для фильтров
- Popup — справа от карты

### Desktop (> 1024px)
- Полноценный layout
- Popup — overlay на карте
