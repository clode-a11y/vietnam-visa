## Admin Pages

### Dashboard `/admin`

- Overview stats
- Recent viewing requests
- Quick links

### Districts `/admin/districts`

- List all districts
- Add / Edit / Delete / Reorder

### Apartments `/admin/apartments`

- List all apartments
- Filters by status, district
- Add / Edit / Delete

### Apartment Form `/admin/apartments/new` & `/admin/apartments/[id]/edit`

- All apartment fields
- Image upload with drag-n-drop
- Amenities checkboxes
- Map pin selection

### Viewing Requests `/admin/requests`

- List all requests
- Filter by status
- Mark as contacted / completed

### Amenities `/admin/amenities`

- Manage amenity list
- Add / Edit / Delete
