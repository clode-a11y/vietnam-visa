---

## Часть 1: Что такое SEO?

**SEO** = Search Engine Optimization (оптимизация для поисковиков)

Это когда сайт настроен так, чтобы Google/Yandex:
- Понимали о чём сайт
- Показывали его в поиске
- Показывали красивое превью

### Шаг 2.1: Перейди в проект

```bash
cd /c/projects/nha-trang/visa

```

### Шаг 2.2: Попроси Claude добавить meta теги

```
Добавь SEO оптимизацию для сайта про визу во Вьетнам.

1. Базовые meta теги:
   - title: "Виза во Вьетнам 2024 — Полный гайд для россиян"
   - description: "Всё о визах во Вьетнам: безвизовый въезд 45 дней, E-Visa, виза по прилёту. Пошаговые инструкции и актуальные цены."
   - keywords: виза вьетнам, виза во вьетнам, вьетнам без визы, e-visa vietnam

2. Open Graph (для соцсетей):
   - og:title
   - og:description
   - og:image (нужна картинка 1200x630)
   - og:url
   - og:type: website

3. Twitter Cards:
   - twitter:card: summary_large_image
   - twitter:title
   - twitter:description
   - twitter:image

4. Дополнительно:
   - viewport
   - robots: index, follow
   - canonical URL
   - lang="ru"

Используй Next.js Metadata API.
```

## Часть 6: Проверь результат

### Шаг 6.1: Проверь meta теги

1. Открой сайт
2. Нажми F12 → Elements
3. Найди `<head>` и проверь что есть все meta теги

### Шаг 6.2: Проверь Open Graph

1. Открой https://developers.facebook.com/tools/debug/
2. Вставь URL своего сайта
3. Нажми "Debug"
4. Посмотри как выглядит превью

### Шаг 6.3: Проверь sitemap

Открой: `https://твой-сайт.vercel.app/sitemap.xml`

### Шаг 6.4: Проверь robots.txt

Открой: `https://твой-сайт.vercel.app/robots.txt`

## Проблемы?

### Превью не обновляется
- Используй "Scrape Again" в Facebook Debugger
- Картинка кешируется — подожди или измени имя файла

### Картинка не показывается
- Проверь размер (1200x630)
- Проверь что путь правильный
- Попробуй абсолютный URL

### Sitemap показывает ошибку
- Проверь формат XML
- Проверь что все URL правильные
