## Часть 2: Найди красивый референс

Найди сайт который нравится и переделай свой по образцу.

### Шаг 2.1: Перейди в проект

```bash
cd /c/projects/nha-trang/visa
```

### Шаг 2.2: Запусти Claude

```bash
claude
```

### Шаг 2.3: Попроси найти референсы

```
Помоги найти красивые примеры информационных сайтов или лендингов.

Нужны референсы для сайта про визу во Вьетнам:
- Современный дизайн
- Простой и понятный
- Приятные цвета
- Хорошо на телефоне

Найди 3-5 примеров с ссылками.
```

### Шаг 2.4: Выбери лучший

Открой каждый сайт, посмотри:
- Цвета
- Карточки
- Шрифты
- Мобильная версия

**Выбери один!**

### Шаг 2.5: Переделай свой сайт

```
Мне понравился сайт [ссылка или название].

Переделай мой сайт про визу в похожем стиле:
- Похожие цвета
- Похожая структура
- Похожие карточки/блоки
- Сохрани весь контент

Покажи что изменишь, потом делай.
```

### Шаг 2.6: Проверь результат

```bash
npm run dev
```

Открой http://localhost:3000

Не нравится? Скажи Claude что исправить:
- "Сделай заголовок крупнее"
- "Поменяй цвет кнопок"
- "Добавь отступов"
