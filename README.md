## Как это работает

1. Получи доступ к аккаунту Claude
2. Пройди [ONBOARDING](./ONBOARDING.md)
3. Выбери задачу из [tasks/](./tasks/)
4. Выполняй с помощью Claude Code
5. Создай Pull Request

## Требования

- Windows 10/11
